Descripción y Contextualización

"El curso trata el diseño de códigos de interfaz para distintos medios y soportes digitales, es decir, la capa Front-end dentro de un proyecto digital. Los alumnos aprenderán a desarrollar proyectos digitales enfocándose en cómo estos se componen y construyen a través de un código limpio, claro y ordenado, utilizando lenguajes tales como HTML, CSS, SASS, Javascript entre otros."

Objetivos

"Conocer y utilizar las herramientas técnicas para la implementación de proyectos digitales. Comprender las posibilidades técnicas y particularidades del medio digital. Comprender las relaciones entre contenidos, datos e interfaz."

Contenidos o Unidades de Aprendizaje

  • Introducción al diseño front-end
  • HTML
  • CSS
  • Tipografías y modelo de caja
  • Técnicas de diagramación
  • JavaScript básico
  • Preprocesadores
  • Accesibilidas
  • SEO
